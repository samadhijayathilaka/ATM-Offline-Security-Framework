# ATM Offline Security Framework

## 📌 Overview
This project proposes an enhanced offline authentication system for ATM machines using QR codes and cryptographic techniques.

## 🔐 Features
- QR-based authentication
- Offline transaction support
- Token-based verification
- Secure hashing (SHA-256)
- Optional NFC integration

## 🛠 Technologies
- Android (Java)
- ASP.NET (C#)
- SQLite / MongoDB
- ZXing (QR Scanner)

## 📊 Security Focus
- Prevents replay attacks
- Reduces card skimming risks
- Works in low-network environments

## 🚀 Future Improvements
- Token expiry validation
- Advanced threat detection
